# Milestone 1

## Project Overview

Milestone 1 is the first phase of the project. In this milestone, the basic application setup, environment configuration, and core functionalities were implemented. The objective is to establish a working foundation that can be enhanced in future milestones.

## Features Implemented

- Project initialization
- Environment setup
- Data loading and preprocessing
- Basic user interface
- Core application logic
- Initial testing and validation

## Technology Stack

- Python 3.x
- Jupyter Notebook / Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## How to Run the Notebook

1. Clone or download this repository.
2. Open the project in Jupyter Notebook or Google Colab.
3. Install the required libraries if needed.

```bash
pip install pandas numpy matplotlib scikit-learn
```

4. Open the notebook file (.ipynb).
5. Run all the cells sequentially from top to bottom.
6. Verify that the outputs are generated successfully.
